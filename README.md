# 🎮 Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 🕹️ How to Play
- Player X starts first
- Click on empty boxes to place X or O
- First player to get 3 in a row wins
- Click restart to play again

## 🚀 Features
- Two player mode
- Win detection
- Draw detection
- Restart button

## 📌 Tech Used
- HTML
- CSS
- JavaScript
